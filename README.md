# Simple To-Do List Manager using Python

## Description
This project is a simple To-Do List application developed using Python. It helps users to manage their daily tasks efficiently. Users can add, view, and delete tasks using a menu-driven interface.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Simple and user-friendly interface

## How to Run
1. Make sure Python is installed on your system
2. Open terminal or command prompt
3. Navigate to the project folder
4. Run the following command:
   python todo.py

## Usage
- Enter the number based on the menu options
- Add tasks and manage them easily
- View the list of tasks anytime
- Delete tasks when completed
